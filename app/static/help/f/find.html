Help on: find

Syntax:find [traps|<object>]
Class:Rogue
Level:8
Minimum:find
Stats:Agility/Perception

Thieves can use this command to toggle on a trap-finding mode. -or- Thieves can use this command to check a room or a specific object to see if it is trapped. Simply specifying "find traps" toggles you into a searching mode. This is slightly more prone to failure than checking specific objects, but can give a rogue peace of mind.
