Help on: fuck

Syntax:socials [<limit>]
Level:any
Minimum:soc

Socials are a method of communicating gestures or expressions to those in the same room as you. They are commands such as `laugh', `cry', `frown', `smile', etc., many of which take a target argument so the gesture or expression can be focused on another user.

The `socials' command displays a list of socials currently defined in the game. If you supply an argument, it only shows socials that begin with that letter.

For on a particular social, try: help <social>

See also: Emote, Say, Quote, Address, Tell, Ask, Reply, Whisper, Communication
