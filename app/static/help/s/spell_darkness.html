Help on: spell darkness

Syntax:cast darkness [<object>]
Class:Magician/Necromancer
Level:2/2
Save:none

If cast upon an object, the darkness spell causes that object to absorb some light in a room attempting to make the room dark. If invoked without an <object>, the spells affects only the room in which it was cast. Plunging a dimly lit cavern should prove to be no problem, but only the highest level casters will be capable of blotting out the noonday sun. The effect lasts for <level of caster> hours.

See also: Spell Light
