Help on: put

Syntax:put <obj> <place>
Level:any
Minimum:put

For putting objects in containers. <obj> can be replaced by "all" here so that you can put all your stuff in a bag (put all bag). If you just want to put the object in the room, use "drop." The object must be in your inventory; if you are currently wearing it, you must "remove" it first.

See also: Get, Take, Give, Drop
