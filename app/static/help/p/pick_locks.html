Help on: pick locks

Syntax:pick <container>|<door name>
Class:Rogue
Level:6
Minimum:pick
Stats:Agility/Focus

This is a rogue's skill that allows the player to open locked objects without having a key. Some locks are harder to pick than others, some are nearly impossible. A rogue must practice this skill somewhat before it can be used.

See also: Rogue, Unpick
