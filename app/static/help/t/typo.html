Help on: typo

Syntax:typo <string>
Level:any
Minimum:typo

The typo command is used for reporting typographical errors and misspellings. There shouldn't be any but then again there shouldn't be any in textbooks either but they are found all the time. <string> can be up to 160 characters in length. Stand in the room with the typo if the typo is in a room description.
