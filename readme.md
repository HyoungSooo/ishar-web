# [isharmud.com](https://isharmud.com/)

This is the code for the website for [Ishar MUD](https://isharmud.com/)!

It depends on a combination of:
- HTML
- JavaScript
- CSS
- Python
- Flask
- Flask-Login
- Flask-SQLAlchemy
- Flask-WTForms
- Jinja2
- MySQL
- nginx
- uwsgi

You should absolutely check out [the website](https://isharmud.com/)!
